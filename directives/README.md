# Directives

Questa directory contiene le SOP (Standard Operating Procedures) in formato Markdown.

Ogni direttiva definisce:
- **Obiettivo**: cosa deve essere raggiunto
- **Input**: dati o informazioni necessari
- **Strumenti**: script da `execution/` da utilizzare
- **Output**: risultato atteso
- **Casi limite**: situazioni particolari da gestire
